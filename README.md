# CSS Grid Generator - React

Generador visual de CSS Grid convertido a React.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

La aplicación se ejecutará en `http://localhost:3000`

## Construcción

```bash
npm run build
```

## Características

- Generador visual de CSS Grid
- Arrastrar y soltar para crear elementos
- Redimensionar elementos
- Mover elementos existentes
- Generación automática de código HTML y CSS
- Interfaz moderna y responsive
